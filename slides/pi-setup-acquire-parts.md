note:
- Must Haves
    - Raspberry Pi (2 Model B+)
    - MicroSD card - 4 Gb min, class 4
    - 5V micro USB power cable (with wall plug)
    - USB Keyboard
    - USB mouse
    - HDMI Monitor and cable
- Nice to Haves
    - USB Wifi dongle
    - Raspberry Pi camera module
- For Blinky
    - Breadboard
    - Female to male jumper wires
    - Male to male jumper wires
    - 330k ohm resistor
    - 5mm LED
